# 📊 SQL Sales Data Analysis

## 🚀 Project Overview
This project analyzes sales data using SQL to find useful business insights.

---

## 🎯 Objectives
- Calculate total sales  
- Identify top customers  
- Analyze category-wise sales  

---

## 🛠 Tools Used
- SQL  
- Superstore Dataset  

---

## 📊 Analysis Performed

### 🔹 Total Sales
Calculated total revenue from all orders.

### 🔹 Top Customers
Identified top 5 customers based on sales.

### 🔹 Category-wise Sales
Analyzed which category performs best.
